# Digital Logic Design Projects

## Description
Collection of Verilog-based digital design projects implemented and simulated using Vivado.

## Projects Included
- 32-bit Counter
- BCD to 7-Segment Display Converter
- 8-Digit Display Controller
- Top Module Integration

## Tools Used
- Verilog HDL
- Xilinx Vivado
